# 📊 QA Monitoring Dashboard

## 실행 방법

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
# 앱과 같은 폴더에 `통합_문서1.xlsx` 파일을 두고 실행
streamlit run app.py
```

### 3. Streamlit Cloud 배포
1. GitHub에 이 폴더를 업로드
2. `통합_문서1.xlsx` 파일도 함께 업로드 (혹은 Google Sheets 연동으로 교체)
3. [streamlit.io/cloud](https://streamlit.io/cloud) 에서 배포

---

## 대시보드 구성

| 페이지 | 내용 |
|--------|------|
| 🏠 개요 | 전체 KPI 요약, 채널별 점수 추이, 상담사 요약 테이블 |
| 📈 점수 트렌드 | 기간별·상담사별 점수 변화, Box Plot |
| 🗂 문의유형 분석 | 유형×상담사 히트맵, 선버스트 차트 |
| 📋 항목별 분석 | 항목 달성률, 시계열, 상담사×항목 히트맵 |
| ⚠️ 감점 분석 | 감점 빈도, 패턴, 사유 텍스트 분석 |
| 👤 상담사 종합평가 | 레이더 차트, 개인 점수 트렌드 |
| ✅ 이행률 분석 | 만점 달성률 기간별·상담사별·항목별 |
| 🔍 평가자 분석 | 엄격도 비교, 점수 편향 분석 |
| 💬 코멘트 NLP | 키워드 빈도, KMeans 클러스터링, 테마 분석 |
